Curso Introdutório de Looker Studio para Usuários de Planilhas

Transforme seus dados de planilhas em dashboards interativos e profissionais. Aprenda a criar visualizações impactantes que contam a história por trás dos números.

O que você aprenderá

  • Conectar planilhas Google e preparar dados
  • Compreender dimensões e métricas
  • Criar tabelas, gráficos e cartões de pontuação
  • Adicionar controles de filtro interativos

1. Introdução e Preparação

1.1 O que é o Looker Studio?

O Looker Studio, anteriormente conhecido como Google Data Studio, é uma ferramenta de business intelligence (BI) e visualização de dados, desenvolvida pelo Google, que permite aos usuários criar relatórios e dashboards interativos e personalizáveis de forma gratuita [24]. A plataforma se destaca por sua capacidade de se conectar a uma ampla variedade de fontes de dados, incluindo produtos do próprio Google como Google Sheets, Google Analytics e Google Ads.

Principais características do Looker Studio:

  • +800 fontes de dados: Conecte-se a praticamente qualquer fonte de dados através de mais de 630 conectores de terceiros
  • Interface intuitiva: Transforme dados brutos em visualizações compreensíveis e atraentes
  • Baseado na web: Facilita a colaboração em tempo real e o compartilhamento de relatórios com stakeholders
  • Atualização automática: Relatórios que se atualizam automaticamente, garantindo informações sempre atualizadas

1.2 Benefícios para Usuários de Planilhas

Visualizações Avançadas

Crie dashboards visualmente atraentes e profissionais que superam as opções básicas de gráficos disponíveis nas planilhas [15].

Automação de Dados

Configure atualizações automáticas a partir da fonte de dados conectada, economizando tempo e reduzindo erros humanos [15].

Colaboração Aprimorada

Compartilhe relatórios facilmente com controles de permissão detalhados para visualização e edição [15].

Escalabilidade

Solução mais escalável capaz de lidar com volumes maiores de dados e maior complexidade de análise [15].

1.3 Preparando sua Planilha Google para Conexão

Importante!

O Looker Studio é uma ferramenta de visualização e não de transformação de dados. Ele funciona melhor quando os dados de origem já estão limpos, organizados e prontos para análise [2].

1.3.1 Organizando os Dados em Formato Tabular

Os dados devem ser estruturados em linhas e colunas, onde cada linha representa um registro único (por exemplo, uma venda, uma visita ao site) e cada coluna representa um atributo desse registro [15].

  • • Garanta que a primeira linha contenha cabeçalhos claros e descritivos
  • • Remova linhas de cabeçalho extras e rodapés com totais
  • • Elimine colunas em branco desnecessárias
  • • Evite células mescladas, tabelas dinâmicas ou gráficos na planilha

1.3.2 Limpando Dados e Removendo Duplicatas

A qualidade dos dados de entrada diretamente impacta a qualidade das visualizações geradas. Dados sujos ou inconsistentes podem levar a insights enganosos [2].

Passos de limpeza:
  • • Remover duplicatas (Dados > Remover duplicatas)
  • • Corrigir valores inconsistentes
  • • Tratar células vazias
  • • Validar tipos de dados em cada coluna
Dicas importantes:
  • • Padronize formatos de data e moeda
  • • Use nomes descritivos para abas da planilha
  • • Verifique consistência entre múltiplas abas
  • • Teste a conexão com uma amostra de dados

2. Conectando sua Planilha Google ao Looker Studio

2.1 Criando uma Nova Fonte de Dados

Passo 1: Acessando o Looker Studio

Acesse lookerstudio.google.com e faça login com sua conta Google [25].

Clique no botão "+ Criar" no canto superior esquerdo e selecione "Relatório" [1] [13].

Passo 2: Selecionando o Conector "Google Sheets"

No painel "Adicionar dados ao relatório", localize e selecione o conector "Google Sheets" na lista de conectores do Google [13].

O Looker Studio solicitará que você autorize o acesso ao Google Drive. Essa autorização é necessária para que a ferramenta possa ler os dados da planilha [7].

2.2 Configurando a Conexão

Escolhendo a Planilha e Aba Correta

Após autorizar o acesso, selecione a planilha desejada e a aba (worksheet) que contém os dados que deseja analisar [6].

Se sua planilha contém múltiplas abas com dados relacionados, você precisará criar uma fonte de dados separada para cada aba.

Verificando a Opção "Usar Primeira Linha como Cabeçalhos"

Certifique-se de que a opção "Usar primeira linha como cabeçalhos" esteja marcada. Isso instrui o Looker Studio a usar os valores da primeira linha como nomes de campo [4] [5].

Se esta opção não estiver marcada, o Looker Studio atribuirá nomes de campo genéricos (Coluna A, Coluna B), dificultando a identificação dos campos.

Intervalo de Células Específico (Opcional)

Você pode definir um intervalo de células específico usando a sintaxe padrão (ex: A1:Z100) para importar apenas um subconjunto de dados [4].

Isso é útil para excluir linhas vazias, rodapés com totais ou áreas da planilha que não são relevantes para a análise.

2.3 Verificando e Ajustando os Tipos de Dados

Após estabelecer a conexão, o Looker Studio exibe uma tela de configuração da fonte de dados onde você pode verificar e ajustar os tipos de dados de cada campo importado [12].

Campos de Dimensão (Verdes)

Atributos qualitativos que descrevem ou categorizam os dados (ex: Produto, Região, Data) [11].

  • • Usados para agrupar e filtrar dados
  • • Contêm dados não agregados
  • • Representados por ícones verdes

Campos de Métrica (Azuis)

Valores quantitativos que medem o desempenho (ex: Vendas, Receita, Quantidade) [11].

  • • Resultado de agregações (soma, contagem, média)
  • • Fornecem medidas numéricas
  • • Representados por ícones azuis

Correção de Tipos de Dados

Verifique e corrija os tipos de dados incorretos. Por exemplo, uma coluna de datas interpretada como texto impedirá a criação correta de gráficos de séries temporais [12].

Clique no ícone do tipo de dados e selecione o tipo correto: Texto, Número, Data, Moeda, etc.

3. Fundamentos: Dimensões e Métricas

3.1 O que são Dimensões?

No Looker Studio, uma dimensão é um atributo ou campo qualitativo em um conjunto de dados que é usado para organizar, agrupar ou filtrar as informações [11].

Quem?

Cliente, Vendedor, Usuário

O quê?

Produto, Categoria, Canal

Onde?

Região, Cidade, País

Identificando Dimensões (Campos Verdes)

As dimensões no Looker Studio são representadas visualmente por campos de cor verde no painel de propriedades e no editor de fonte de dados [8] [12].

Exemplos comuns: Nome do Produto, Cidade, Canal de Marketing, Data da Transação.

Exemplos Práticos de Dimensões

Em um relatório de vendas:
  • • Data da Venda: Análise temporal
  • • Produto: Identificação de itens
  • • Categoria: Agrupamento de produtos
  • • Região: Análise geográfica
Em análise de tráfego:
  • • Página: URL visualizada
  • • Fonte/Mídia: Origem do tráfego
  • • Tipo de Dispositivo: Desktop, Mobile
  • • Nome do Evento: Ações do usuário

3.2 O que são Métricas?

Uma métrica é um campo de dados quantitativo que representa uma medida numérica do desempenho ou do impacto de uma dimensão [11].

Quanto?

Receita, Vendas, Lucro

Quantos?

Pedidos, Clientes, Cliques

Com que frequência?

Taxas, Porcentagens

Identificando Métricas (Campos Azuis)

As métricas no Looker Studio são representadas visualmente por campos de cor azul no painel de propriedades e no editor de fonte de dados [8] [12].

As métricas são geralmente o resultado de uma agregação: soma (SUM), contagem (COUNT), média (AVG), etc.

Exemplos Práticos de Métricas

Em um relatório de vendas:
  • • Receita Total: Soma dos valores de venda
  • • Número de Pedidos: Contagem de transações
  • • Ticket Médio: Valor médio dos pedidos
  • • Margem de Lucro: Diferença percentual
Em análise de marketing:
  • • Número de Cliques: Contagem de cliques
  • • Taxa de Conversão: Porcentagem de conversões
  • • Custo por Clique: Média de custo (CPC)
  • • ROI: Retorno sobre investimento

3.3 A Relação entre Dimensões e Métricas

A Fórmula da Análise de Dados

Dimensões (Verdes) + Métricas (Azuis) = Insights Ação

As dimensões fornecem o "o quê" e as métricas fornecem o "quanto" [11].

Como as Métricas Fornecem Insights sobre as Dimensões

As métricas são a lente através da qual as dimensões são analisadas. Elas transformam categorias em fontes de insights acionáveis.

Dimensão: Produto

Categoriza os itens vendidos

Métrica: Receita Total

Mede o desempenho de cada produto

Dimensão: Região

Segmenta por área geográfica

Métrica: Número de Clientes

Compara penetração de mercado

Comparação com Tabelas Dinâmicas em Planilhas

Para usuários de planilhas, a relação entre dimensões e métricas no Looker Studio é análoga às tabelas dinâmicas (pivot tables):

Tabela Dinâmica
  • • Linhas/Colunas: Dimensões
  • • Valores: Métricas
  • • Filtros: Filtros de dimensão
Looker Studio
  • • Dimensões: Campos verdes
  • • Métricas: Campos azuis
  • • Visualizações: Gráficos, tabelas

4. Criando Visualizações: Tabelas e Gráficos

4.1 Adicionando uma Tabela

As tabelas são ideais para exibir informações detalhadas e permitir análise comparativa linha por linha. São uma das formas mais diretas de apresentar dados no Looker Studio.

Inserindo uma Tabela

Clique no ícone "Adicionar um gráfico" na barra de ferramentas, e escolha um modelo de tabela.

Adicionando Dimensões e Métricas

No painel de propriedades, arraste e solte os campos da fonte de dados nas áreas apropriadas:

  • • Dimensões (verdes) → Área de "Dimensão"
  • • Métricas (azuis) → Área de "Métrica"

A ordem dos campos define a hierarquia de agrupamento na tabela.

Personalizando a Apresentação

A guia "Estilo" no painel de propriedades permite personalizar:

  • • Fonte, tamanho e cor do texto
  • • Cores de fundo das linhas
  • • Formatação condicional
  • • Exibição de totais e paginação
  • • Alinhamento do texto

Ordenação de Dados

Você pode ordenar a tabela clicando nos ícones de seta no cabeçalho de qualquer coluna no modo de visualização.

Para definir uma ordenação padrão, use o painel de propriedades para especificar a coluna e direção (crescente/decrescente).

4.2 Criando Gráficos

Os gráficos transformam dados brutos em insights visuais fáceis de entender e interpretar. A escolha do tipo de gráfico certo é fundamental para comunicar sua mensagem de forma eficaz.

Tipo de Gráfico Descrição Exemplo de Uso
Gráfico de Colunas/Barras Compara valores entre categorias Comparar receita por produto ou região
Gráfico de Linhas/Série Temporal Mostra tendências ao longo do tempo Acompanhar evolução de vendas diárias/mensais
Gráfico de Pizza Mostra proporção de partes do todo Distribuição percentual de vendas por categoria

Configurando os Eixos com Dimensões e Métricas

Eixo X (Horizontal)

Geralmente contém uma dimensão que representa as categorias ou o tempo.

Eixo Y (Vertical)

Contém uma métrica que representa os valores quantitativos a serem medidos.

No painel de propriedades, arraste os campos para as áreas "Dimensão" (eixo X) e "Métrica" (eixo Y).

Dimensões de Quebra (Breakdown)

Adicione um nível extra de detalhe arrastando uma segunda dimensão para a área de "Quebra de Dimensão".

Exemplo: Um gráfico de barras com "Mês" no eixo X pode ter "Categoria de Produto" como quebra, mostrando a contribuição de cada categoria para as vendas mensais.

4.3 Utilizando Cartões de Pontuação (Scorecards)

Os cartões de pontuação são componentes visuais simples, mas extremamente eficazes, usados para destacar uma única métrica-chave (KPI) em um dashboard. Eles exibem o valor atual da métrica de forma clara e proeminente.

R$ 125.430

Receita Total

2.847

Clientes Ativos

12.5%

Taxa de Conversão

Inserindo um Cartão de Pontuação

  1. 1. Clique no ícone "Adicionar um cartão de pontuação" na barra de ferramentas
  2. 2. Selecione o cartão no canvas
  3. 3. No painel de propriedades, arraste uma métrica (campo azul) para a área de "Métrica"
  4. 4. Personalize o estilo na guia "Estilo"

Configurando Comparações com Períodos Anteriores

Uma das funcionalidades mais poderosas dos scorecards é mostrar a comparação com um período anterior, adicionando contexto valioso.

Passos: Selecione o cartão de pontuação e marque a caixa "Mostrar comparação" no painel de propriedades. Defina o período de comparação e as cores da seta (verde para aumento, vermelho para diminuição).

5. Adicionando Interatividade com Controles de Filtro

5.1 Filtros de Intervalo de Data

Os filtros de intervalo de data são controles interativos que permitem aos usuários ajustar o período de tempo dos dados exibidos em todos os componentes do dashboard.

Adicionando um Controle de Intervalo de Data

  1. 1. Clique no ícone "Adicionar um controle" na barra de ferramentas
  2. 2. Selecione "Controle de intervalo de data"
  3. 3. Posicione o controle no canvas
  4. 4. No painel de propriedades, selecione a dimensão de data a ser usada

Configurando o Período Padrão

Configure um período padrão para garantir que o relatório sempre abra com uma visão de tempo relevante.

Hoje

Esta Semana

Este Mês

Outras opções disponíveis: Últimos 7 dias, Últimos 30 dias, Este Ano, etc.

5.2 Filtros de Dimensão

Os filtros de dimensão permitem filtrar os dados com base em atributos categóricos como "Produto", "Região" ou "Canal de Marketing". O Looker Studio oferece dois tipos principais:

Lista Suspensa

Menu suspenso que economiza espaço no canvas. Ideal para dimensões com muitos valores únicos.

Lista de Tamanho Fixo

Exibe todas as opções de uma vez. Útil quando há um número limitado de valores.

Adicionando um Controle de Filtro de Dimensão

  1. 1. Clique em "Adicionar um controle" na barra de ferramentas
  2. 2. Escolha entre "Lista suspensa" ou "Lista de tamanho fixo"
  3. 3. Posicione o controle no canvas
  4. 4. No painel de propriedades, selecione a dimensão a ser filtrada

Permitindo Seleção de Múltiplos Valores

Por padrão, os controles permitem selecionar apenas um valor. Você pode permitir a seleção de múltiplos valores para análises mais complexas.

Passo: Selecione o controle de filtro e marque a caixa "Selecionar múltiplo" no painel de propriedades.

5.3 Filtros de Cruzamento (Cross-filtering)

Análise Apontar-e-Clicar

O filtro de cruzamento permite filtrar um relatório inteiro simplesmente clicando em um elemento de uma visualização. Crie uma experiência de análise de dados fluida e intuitiva.

Ativando a Interação de Filtro Cruzado

  1. 1. Selecione o gráfico que deseja tornar interativo
  2. 2. No painel de propriedades, vá para a guia "Configuração"
  3. 3. Na seção "Interações", marque a caixa "Filtro cruzado"

Uma vez ativado, o gráfico se torna um controle de filtro. Cliques em elementos aplicarão filtros a toda a página.

Como Funciona o Filtro Cruzado

Clique em um Elemento

Ex: Clique em uma barra específica em um gráfico de barras

Filtro Aplicado

Um filtro é definido baseado na dimensão e valor do elemento clicado

Atualização Global

Todos os componentes do relatório são filtrados para mostrar apenas dados relacionados

Para remover o filtro, clique no mesmo elemento novamente ou use o botão "Limpar seleção" que aparece no canto superior do relatório.

6. Finalizando e Compartilhando seu Relatório

6.1 Aplicando Temas e Layouts

Escolhendo um Tema Predefinido

Clique em "Tema e layout" no menu do relatório para acessar uma variedade de temas pré-configurados.

• Cores harmoniosas e profissionais

• Fontes e estilos consistentes

• Aplicação com um único clique

Escolhendo o Tipo de Layout

Layout Livre

Controle preciso sobre o posicionamento de cada elemento. Ideal para relatórios em desktop.

Layout Responsivo

Ajusta automaticamente para diferentes tamanhos de tela. Melhor para dispositivos móveis.

Personalizando Cores e Fontes para Alinhar com sua Marca

No painel de temas, clique em "Personalizar" para criar um tema que reflita a identidade visual da sua marca.

Personalização de Cores:
  • • Defina cores primária e secundária
  • • Escolha cores de destaque
  • • Use códigos hexadecimais para precisão
Personalização de Fontes:
  • • Escolha fontes do Google Fonts
  • • Defina tamanhos para títulos e texto
  • • Mantenha consistência visual

6.2 Adicionando Títulos e Anotações

Inserindo Caixas de Texto para Contexto

Use a ferramenta de texto para adicionar contexto ao relatório:

  • • Títulos principais: Descrevam o propósito do dashboard
  • • Subtítulos: Explique seções específicas
  • • Rodapés: Informações importantes como data de atualização
  • • Notas explicativas: Destaque insights e anomalias

A formatação do texto (fonte, tamanho, cor, alinhamento) pode ser ajustada na guia "Estilo" do painel de propriedades.

Adicionando Formas e Imagens para Melhorar o Design

Formas:
  • • Retângulos e círculos para agrupar visualmente componentes
  • • Linhas divisorias para separar seções
  • • Setas para indicar fluxos ou relações
Imagens:
  • • Logotipo da empresa no cabeçalho
  • • Ícones para representar seções visualmente
  • • Imagens de fundo para design atraente

6.3 Salvando e Compartilhando seu Relatório

Salvamento Automático do Looker Studio

O Looker Studio possui um sistema de salvamento automático. Todas as alterações são salvas em tempo real, sem a necessidade de um botão "Salvar" tradicional.

Para reverter alterações, use Ctrl+Z (ou Cmd+Z no Mac) ou acesse o histórico de versões no menu "Arquivo" > "Histórico de versões".

Compartilhando o Relatório com Outros Usuários

  1. 1. Clique no botão "Compartilhar" no canto superior direito
  2. 2. Adicione os endereços de e-mail das pessoas
  3. 3. Defina o nível de permissão
  4. 4. Opcionalmente, adicione uma mensagem ao convite

Níveis de Permissão

Visualizador

Pode ver, interagir com filtros e exportar dados, mas não pode fazer alterações

Editor

Pode modificar visualizações, adicionar componentes e alterar o layout

Proprietário

Controle total, incluindo a capacidade de excluir o relatório

Alternando entre os Modos de Edição e Visualização

O Looker Studio possui dois modos principais: Modo de Edição e Modo de Visualização.

Use o botão "Visualizar" para ver o relatório como os usuários finais o verão. Clique em "Editar" para voltar ao modo de construção. Teste sempre no Modo de Visualização para garantir uma experiência do usuário fluida.
